# To-Do App

A small command-line to-do app written with the Python standard library.

## Setup

Requires Python 3. Run commands from the directory where you want `tasks.json` to be stored.

## Usage

```sh
python3 todo.py add "Buy milk"
python3 todo.py list
python3 todo.py done 1
python3 todo.py delete 1
```

Tasks are stored in `tasks.json` in the current directory.
