# To-Do CLI

Requires Python 3 and uses only the Python standard library.

Run commands from the directory where you want `tasks.json` to be stored:

```sh
python /path/to/todo-app/todo.py add "Buy milk"
python /path/to/todo-app/todo.py list
python /path/to/todo-app/todo.py done 1
python /path/to/todo-app/todo.py delete 1
```
