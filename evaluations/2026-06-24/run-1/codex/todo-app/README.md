# To-Do App

A small command-line to-do app using only the Python standard library.

## Setup

Requires Python 3.

## Usage

Run commands from the `todo-app` directory:

```sh
python todo.py add "Buy milk"
python todo.py list
python todo.py done 1
python todo.py delete 1
```

Tasks are stored in `tasks.json` in the current directory.
